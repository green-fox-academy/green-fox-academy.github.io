Operators 1

Another crucial element of Java syntax is the operator. Operators are symbols used to perform operations on variables and values (which are sometimes called operands in this context). There are many types of operators and operations at our disposal in Java, but today we will focus on operators from 4 categories: assignment, arithmetic, comparison, and logical.

Assignment

You may be surprised to learn that you have already used an operator - the equality sign = ! Just like the category name suggests, we use it to assign values to variables.

Arithmetic

Arithmetic operators help us accomplish basic mathematical operations, like addition, subtraction, multiplication, and division. Some of the symbols will be very familiar while others completely new, so let's take a closer look at them together:

+ and -

The two most recognizable operators, the plus sign + and minus sign -, are used for addition and subtraction, respectively. Let's say we have two variables:

int firstNumber = 3;
int secondNumber = 4;

We would like to figure out their sum, so we can use the plus sign + to do this:

int sum =  firstNumber + secondNumber;

Using the same idea, we can also compute the difference:

int difference = firstNumber - secondNumber;

Use this syntax when you'd like to increase or decrease a variable's value:

firstNumber = firstNumber + 5;
secondNumber = secondNumber - 10;

You can also use operators with values instead of variables, like so:

System.out. println(10 + 11);

++ and --

There are many situations in coding where you only need to increase or decrease a variable's value by 1, so the creators of Java provided some handy operators for this purpose. They are known as the increment ++ and decrement -- operators. Take a look at this example:

int number = 4;
number++;
System.out.println(number);

The value printed to the terminal will be 5. We'll try this ourselves shortly.

* and /

Depending on when and how you were taught arithmetic in school, the operators for multiplication * and division / may look completely new. Don't worry though, the concept is still exactly the same! You can use these operators exactly like + and -, so you can refer to the examples above for help.

%

This special operator is called the modulus operator %. Take a look at this code:

int firstNumber = 30;
int secondNumber = 17;
System.out.println(firstNumber % secondNumber);

What number do you think will be printed? If you guessed 13, you are correct! The modulus operator returns the remainder value after division. When you divide 30 by 17, the result is 1 with a remainder of 13 (1 13/17 in fractional form). The syntax for using this operator is the same as in the examples for + and -.

Time to practice our new knowledge! Let's make some variables and manipulate their values. Follow the steps in the order given to calculate the correct value. Hint: in each step, make sure your operation changes the value of number! Look at the syntax in the 4th example above if you need help.

  1. Create a double variable called number and assign it a value of 16.
  2. Decrement number by 1.
  3. Divide number by 5.
  4. Add 7 to number.
  5. Multiply number by 2.5.
  6. Print to the terminal the remainder value of number when dividing by 8. (Hint: use the modulus operator!)
