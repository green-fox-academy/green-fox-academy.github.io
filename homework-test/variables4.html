Variables 4

Non-numeric

boolean

Variables of type boolean can only have 2 values: true or false. The boolean type exists because Boolean algebra, which is a fascinating topic, has many important applications in computer programming. We'll learn more about it in the "operators" section.

When declaring boolean variables, it often makes sense to give them names that start with "is" or "has," like isVerified or hasOpened. This is because true and false can be answers to a yes/no question, such as "Is this account verified?" or "Has the recipient opened the email?"

Did you notice that all variable names start with a lowercase letter, and each new word in the name starts with an uppercase letter? This naming convention is called "camel case" and it's the standard for Java. Create good coding habits now by following it! During the course, mentors will give you more help and support with this.
