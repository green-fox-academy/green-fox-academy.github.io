Variables 2

Numeric

We’ve already learned a little about one numeric type, int. You can read more about integers here. Java has 6 different basic numeric types, but today we will only focus on two: int and double.

int

int type variables can only store whole numbers between -2,147,483,648 to 2,147,483,647. If you try to create an int variable with a non-integer value, like -24.574 or 398.5, your code will not run.

double

When you need to work with fractional values (called floating point numbers in computer programming lingo), you can use the double type for that. A double type variable can store any value between 1.7976931348623157 x 10^308 and 4.9406564584124654 x 10^324.
