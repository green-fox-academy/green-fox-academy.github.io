If...Else Statements

In the previous section you learned how to use operators to write conditions based on Boolean algebra. We practiced printing the results to the terminal, so now let's try using them to control the flow of our code.

We can accomplish this by using if- and else-statements. Take a look at this code:

if (x < y) {
  System.out.println("the condition is true.");
}

Do you see the condition inside the if-statement's parentheses ( )? When you combine conditions with the syntax for if-else statements, you can create conditional statements. You would read the conditional statement in English as, "If x is less than y, then print "the condition is true." If-statement syntax comes from if-then statements, which are statements that have a hypothesis ("if") followed by a conclusion ("then").

When a condition inside the if-statement is true, the code inside of the curly braces (also called a block) will execute. If the condition is false, then the program will skip over the block and continue executing the code below.

If you would like certain code to run when the condition is false, you need to add an else-statement:

if (x < y) {
  System.out.println("The condition is true.");
} else {
  System.out.println("The condition is false.");
}

If you ran the code below:

class Main {
  public static void main(String args[]) {
    int number1 =  14;
    int number2 = 10;

    if (number1 < number2) {
      System.out.println("number1 is smaller than number2");
    } else {
      System.out.println("number2 is less than or equal to number1");
    }
  }
}

You’d see the following output:

number2 is less than or equal to number1

We can see only 1 printed statement. That's because when the conditional statement is true, only the if-statement block executes. If it is false, only the else-statement block executes.

Now it's your turn - translate this English sentence into a condition for the if-statement: x is greater than y and x is equal to 10.

The condition is true.
