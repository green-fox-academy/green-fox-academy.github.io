Print Statements 1

A handy way to literally see the results of your code's work is to use print statements. Print statements show output on your screen (sometimes called the output terminal or console). Thankfully, Java has some built-in methods that we can use to accomplish this! Take a look at the code below:

System.out.println("Hello world!");

System is a built-in class in Java that provides functionality for working with inputs and outputs (among other things). So to use the functionality for printing, we need to type .out and then .println(); after System. Notice that the line ends with a semicolon ;. Semicolons are required to indicate the end of a logical statement - without them your code won't compile!

Let's try it out! Type System.out.println("Green Fox"); in the window below and click "run" to see the output!
