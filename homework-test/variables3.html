Variables 3

Non-numeric

Java contains 3 non-numeric types that are an important tool in every developer’s skill set: char, String, and boolean.

char

char is short for “character”. A character in computer programming lingo is simply 1 letter, digit, or symbol, like a, 4, or @. Variables of type char can only store 1 character at a time. When you declare a char type variable, you must use single quotes ' ' when assigning the value, like this:

char middleInitial = 'B';

String

A string in OOP languages refers to a collection of characters that you can work with as a single variable. This type is perfect to use when you want to work with information such as names, addresses, etc. When you declare String type variables, you must use double quotes " " when assigning the value, like this:

String companyName = "Green Fox Academy";

Notice that the first letter in String is capitalized, unlike the other types we’ve learned about so far!

  • Declare a char type variable called exclamationPoint and set it to the correct symbol.
  • Declare a String type variable called czechCapitalCity and set it to the correct value. Hint: in English, proper nouns (including city names) always start with an uppercase letter.
