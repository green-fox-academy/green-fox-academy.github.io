Operators

Logical

Java’s logical operators are implementations of the Boolean algebraic operations AND &&, OR ||, and NOT ‘!’. As we mentioned before, Boolean algebra is a fascinating topic that has many important applications in computer programming. Don’t worry, you don’t need to become an expert to pass the course or become a developer, you just need to understand how these operators work. If you’d like to know more, you can check out a Boolean algebra tutorial here.

Comparison and logical operators are often combined to create logical statements. Comparison operators are used to create conditions (remember, the result is always true or false), and we can use logical operators to evaluate 2 conditions, which gives us another boolean result.

&&

The AND && operator evaluates 2 conditions (one on each side of it). Take a look at this code:

int x = 7;
int y = 10;
System.out.println(x < y && x != y);

The logical statement inside println is read as, “x is less than y and x does not equal y.”

Let’s break this statement down together: first, we need to evaluate x < y. Since 7 is less than 10, the left side is true. Then, we evaluate the right side and know that 7 does not equal 10, so it is also true. Using Boolean algebra’s rules, true AND true results in true. So, the print statement will show us true in the terminal.

But what happens when one side (or perhaps both) is false? According to Boolean algebra’s rules, when at least 1 expression is false, the overall result is false. You can see this information visualized below in the truth table.

To summarize, AND && statements evaluate to true if both conditions are true. Otherwise it evaluates to false.

Truth Table (A && B)

A B Result
true true true
true false false
false true false
false false false

||

The OR || operator evaluates 2 conditions, one on each side of it, just like the AND && operator. The syntax for using OR || is the same as AND &&:

x > y || x == y

However, unlike AND && expressions, OR || statements evaluate to true when at least one of the conditions is true. Look at the truth table below to see all possible results.

Truth Table (A || B)

A B Result
true true true
true false true
false true true
false false false

!

The NOT ! operator is used to invert a boolean value. You can see in the truth table below that NOT ! turns true into false and vice versa. This operator is useful when you want to invert the result of an entire logical statement, for example:

x > y || !(x == y && x > 0)

Truth Table (!A)

A Result
true false
false true
