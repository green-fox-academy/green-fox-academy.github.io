Command Line

Before you start this exercise, please go through the materials of the Materials - Command Line chapter in the LMS. This exercise helps putting what you learn from the videos into practice.

You can find an emulation of a Linux terminal below. This is very similar to what you see when you log in to a server, which is something that you will occasionally need to do as a web developer. Wait for the /root # command prompt to appear, and you can start typing the commands.

The commands to use are cd, cp, ls, mkdir, mv, pwd, rm and touch. If you don’t remember what they do, revise the videos, or execute the commands with the --help option, which will give you a brief explanation. Don’t forget that you can browse the recently used commands with the up and down arrows, and auto-complete commands with the tab key.

Note: this exercise is not tablet-friendly, you need to use a physical keyboard.
