Variables 1

A variable is a way to store information in a computer program. This is useful because it allows us to reference the value and use it in our code to do different actions and calculations. Let's take a closer look at a variable declaration:

int fingerCount = 10;

The first word, int, tells us what type of variable it is (int means it's an integer, or a whole number). The next word, fingerCount, is the name. Variable names should be descriptive, otherwise it will be difficult to understand what the value means. Finally, the last part = 10; is the value assignment. This syntax means the variable called fingerCount is given a value of 10. Since that is the end of the logical statement, we finish it with a ; .

Let's practice! Create a variable of type int called daysInWeek and set it to the correct value. Notice how we print the value to the terminal by putting the variable name inside of the print method's parentheses ( ).
